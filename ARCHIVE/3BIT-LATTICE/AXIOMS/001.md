# AXIOM 001

**Bits:** 0 0 1  
**Origin/Mirror:** Origin  
**Generation/Constraint:** Generation  
**Self/Other:** Other  

**Question:**  
*What is the other that generates from origin?*

**Inverse Axiom:** 110  
**Domain:** Other‑Origin Generation  

**Notes:**  
This asks about the first relationship: the other that emerges alongside the self from the same origin. Generation without constraint, but now there is a second pole. The self sees itself through this other.
