# Reconstructed one-page version

## Restitution-Funded UBI Framework

Universal basic income can be modeled as a restitution mechanism rather than a purely fiscal program.

The core hypothesis is that multiple forms of extraction across society share a common structure: value is taken from individuals or groups through unequal access, labor asymmetry, exclusion, biased systems, uncompensated data use, or regulatory advantage. These harms are typically handled in isolation, but they can be represented in a shared claims framework.

The framework has six steps:

```text
harm domain
→ claim class
→ evidence model
→ valuation method
→ pooled restitution ledger
→ public distribution mechanism
```

Candidate domains include disability access failures, wage theft, discriminatory finance, housing exclusion, healthcare extraction, educational underinvestment, tax avoidance, and uncompensated AI-related data or pattern extraction.

In this model, UBI is not treated as charity. It is treated as a distribution layer over a validated restitution pool. The purpose of the framework is not to assert final damages in advance, but to provide a common ontology that can later support measurement, prioritization, auditing, and payout design.

The prototype requirement is modest: define the ontology, claim classes, and evidence structure first. Quantification and enforcement can come later.
