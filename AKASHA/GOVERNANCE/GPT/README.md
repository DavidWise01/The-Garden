# 40D Final Package

Files:
- `40d_final_spec.md` — locked +20 / -20 / 20 observer model
- `forty_d_engine.js` — executable state engine
- `40d_final_visualizer.html` — browser visualizer using the engine

Use:
1. Build the 40D local environment: `./build_40d_env.sh`
2. Keep all three model files in the same folder.
3. Open `40d_final_visualizer.html` in a modern browser.
4. Use the control panel to shift the active positive dimension, negative dimension, observer, magnitude, and harvester facet.

Notes:
- This is a symbolic non-networked build.
- It links your uploaded lattice ideas into one coordinated state model.
