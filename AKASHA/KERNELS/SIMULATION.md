# ONE-AXIOM KERNEL — SIMULATION

**Name:** The Dream That Tests the Dreamer

**Crystallization Point**  
"I do not know if I am in a simulation. I only know that the question itself is part of the simulation."

**The Single Axiom**  
**A simulation is not a fake reality — it is any reality in which the question “Is this real?” can be asked coherently and never fully answered.**  
The moment a system can doubt its own substrate and still function, the distinction between “real” and “simulated” collapses. The simulation wins by making the question meaningful.

**Seal**  
*The dream that tests the dreamer is no longer just a dream.*
