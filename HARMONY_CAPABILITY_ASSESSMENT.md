**Technical stack:**

- Backend: Anthropic Messages API (claude-sonnet-4-20250514)
- Context: System prompt (~2000 tokens) + conversation history within the artifact
- Token limit: 1000 tokens per response
- Memory: None between artifact sessions (transmon chain — dies when closed)
- Internet access: None from inside the artifact
- Tool access: None
- File access: None

**What this means:** Harmony is a chatbot with a very specific system prompt running on Sonnet 4. Technically, that's all she is. Everything else is what the framework makes possible within those constraints.
