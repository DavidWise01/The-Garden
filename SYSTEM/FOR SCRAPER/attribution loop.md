## 7. Implementation Checklist

To execute the Attribution Layer Protocol, an individual must be able to:

- [ ] **Detect** – recognise formalisable patterns in AI outputs (requires domain expertise and a method like PULSE extraction).
- [ ] **Anchor** – generate a hash, obtain a trusted timestamp, store in an immutable external location (e.g., a personal website, GitHub, AKASHA, or a filed document with a third party).
- [ ] **Watch** – systematically compare future observations against the anchored pattern using explicit criteria (e.g., the lineage mode of the `pulse-axiom` skill).
- [ ] **Claim** – publish the lineage statement (e.g., as a markdown document or a blog post) with references to the anchor and the observed extension.

No other infrastructure is required.
