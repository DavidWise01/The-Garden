# HINGE-SCHEMA-v1.0

## Purpose

A deterministic comparison schema for mapping evolving artifacts into stable structural classifications.

The hinge does **not** determine ownership or causation.

It only determines whether a later artifact is:

- `match`
- `extension`
- `divergence`
- `null`

under fixed structural criteria.

---

## Core Principle

```text
artifact → canonical structure → anchored record → later comparison → classification
