# 3-2-1 Compression Guide

**How to use the 3/2/1 Compressor tool**

### Purpose
This tool helps turn raw exploratory notes into clean 3/2/1 form, and expand compressed ideas back out.

It follows the natural law:
- **3** = Wide attention / broad idea
- **2** = Narrowed refinement
- **1** = Singular core point
